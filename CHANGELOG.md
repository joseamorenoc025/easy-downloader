# Registro de Cambios (Changelog)

Todos los cambios notables en este proyecto se documentarán en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto se adhiere a [Versionado Semántico](https://semver.org/lang/es/).

## [1.0.0] - 2026-05-23

### Añadido
- Interfaz gráfica principal desarrollada con `CustomTkinter`.
- Integración con `yt-dlp` para descargas de video y extracción de audio.
- Sistema de cola de descargas secuencial.
- Soporte para formatos MP4 y MP3.
- Selector de resolución de video.
- Interfaz reactiva con visualización de progreso y tamaño estimado.
- Documentación inicial incluyendo README, CONTRIBUTING, LICENSE y plantillas de Issues.
- Flujo de trabajo de GitHub Actions para compilación en Windows.
